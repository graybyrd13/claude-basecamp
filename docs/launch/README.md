# Launch kit

Ready-to-paste posts for announcing Basecamp. Post them yourself from your own accounts — in this order, a day or two apart:

1. **awesome-claude-code PR** (`awesome-claude-code.md`) — quiet, evergreen discovery channel. Do this first.
2. **r/ClaudeAI** (`reddit.md`) — friendliest audience, fastest feedback loop.
3. **Show HN** (`show-hn.md`) — post on a weekday morning US time. Stay in the comments for the first 2 hours; HN rewards responsive authors.
4. **dev.to article** (`devto.md`) — longer shelf life, good SEO for "claude code dashboard".

Notes:
- Don't post to Stack Overflow — project promotion is off-topic there and gets removed.
- Update the token numbers in the posts with your real Stats page numbers before posting; concrete numbers are the hook.
- Add screenshots before posting: Repositories page, a manager chat, and the Stats heatmap are the three that sell it.
